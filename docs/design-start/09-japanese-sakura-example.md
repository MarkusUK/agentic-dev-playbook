# 09 - Japanese Sakura Example

Use this as a ready-made example.

## Starter prompt

```text
I want to design a new app/product.

Project idea:
A mindful habit-tracking app that helps users build gentle daily routines without guilt, streak pressure, or noisy gamification.

Target users:
Busy adults who want calm structure, especially people interested in wellness, journaling, mindfulness, and low-pressure self-improvement.

Primary goal:
Help users create, track, and reflect on small daily rituals.

Theme / brand direction:
Japanese sakura theme. Calm, elegant, seasonal, warm, minimal, premium but approachable. Avoid anime clichés or overly decorative pink interfaces.

Platform:
Android first, later iOS and web.

Tech stack if known:
Android Jetpack Compose.

Design constraints:
- Must be accessible
- Must support light and dark mode
- Must be buildable with Jetpack Compose
- Should avoid generic AI-looking UI
- Should not add unnecessary dependencies
- Should feel coherent as a brand system, not just a pretty screen

Available tools:
- Figma MCP: yes
- Google Stitch MCP: yes
- 21st.dev/Magic MCP: yes
- Rive: possible later
- Existing design system: no, starting fresh

Task:

Do not write code yet.

Act as a product designer, brand strategist, and senior mobile app designer.

First, produce a design discovery brief with:

1. Product summary
2. Target audience
3. User emotions we want to create
4. Brand personality
5. Visual principles
6. Accessibility considerations
7. Design risks or clichés to avoid

Then create 3 distinct design directions:

1. Minimal Sakura
2. Kyoto Editorial
3. Night Hanami

For each direction include:

- Name
- One-sentence concept
- Brand personality
- Colour palette with hex values
- Typography direction
- Spacing/layout style
- Component style
- Icon/illustration style
- Motion/animation style
- Example home screen description
- Example onboarding screen description
- Example settings/profile screen description
- Pros
- Cons
- Best fit

Then compare the 3 directions in a table.

Finally, recommend one direction and explain why.

Do not use external MCP tools yet unless I explicitly ask you to.
```

## Useful ingredients

```text
Colours:
- warm off-white
- sakura blush
- muted rose
- deep plum
- charcoal ink
- matcha/sage accent
- lantern amber accent

Shapes:
- soft rounded cards
- subtle petal-like curves
- lots of breathing room

Motion:
- gentle petal drift
- soft screen transitions
- progress blooming over time
- reduced-motion fallback

Typography:
- clean modern sans for UI
- optional elegant serif only for headings/brand moments
```

## Avoid

```text
- too much pink
- anime clichés unless that is the actual audience
- random cultural motifs
- low-contrast pastel text
- mixing too many typefaces
- overly busy blossom backgrounds
```
