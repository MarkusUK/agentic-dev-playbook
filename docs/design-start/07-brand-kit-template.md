# 07 - Brand Kit Template

Use this once a direction is chosen.

## Copy/paste prompt

```text
Create a complete brand kit for this selected design direction:

[Paste selected direction]

Include:

## Brand summary
- One-sentence brand essence
- Personality
- Voice/tone
- Visual principles

## Logo direction
- Concept
- Shape language
- Wordmark direction
- App icon direction
- What to avoid

## Colour system
- Primary
- Secondary
- Accent
- Background
- Surface
- Text primary
- Text secondary
- Border
- Error
- Warning
- Success
- Light mode tokens
- Dark mode tokens

Use hex values and explain usage.

## Typography
- Font direction
- Heading scale
- Body scale
- Caption/meta text
- Button text
- Accessibility notes

## Layout
- Spacing scale
- Grid or layout rules
- Density
- Card style
- Radius scale
- Shadow/elevation

## Components
- Buttons
- Cards
- Inputs
- Navigation
- Tabs
- Lists
- Empty states
- Dialogs/sheets
- Toasts/snackbars

## Iconography
- Stroke/fill style
- Size rules
- Metaphor rules

## Illustration/images
- Style
- Allowed motifs
- Forbidden motifs

## Motion
- Transition style
- Microinteractions
- Loading states
- Reduced-motion fallback

## Accessibility
- Colour contrast
- Touch targets
- Text scaling
- Screen reader labels
- Motion sensitivity

## Platform implementation notes
- Android
- iOS
- Web

Do not write production code yet.
```
