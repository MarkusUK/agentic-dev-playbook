# 00 - Overview Workflow

## Recommended sequence

```text
1. Describe the product idea.
2. Generate a design discovery brief.
3. Generate 3 brand/design directions.
4. Compare and choose one.
5. Use Figma/Stitch/21st.dev/Magic/Rive only if helpful.
6. Produce a brand kit.
7. Produce screen specs.
8. Optionally produce interactive mockups.
9. Translate chosen design into implementation plan.
10. Implement one small slice.
11. Review diff with second agent.
```

## Tool roles

```text
Claude / ChatGPT / Codex:
- product thinking
- brand exploration
- screen IA
- design critique
- implementation planning
- code review

Figma:
- visual design source of truth
- design components
- screen layout
- design handoff

Google Stitch:
- quick design exploration
- visual direction alternatives
- early UI ideation

21st.dev / Magic:
- component and layout inspiration
- shadcn-style ideas
- UI polish ideas

Rive:
- animation concepts
- interactive motion
- state machine based animations

React/Tailwind interactive mockups:
- fast clickable prototypes
- early flow testing
- layout experimentation
```

## Recommended rule

Do not jump straight from idea to production code.

First create:

```text
- brand direction
- screen list
- design system tokens
- component list
- interaction notes
- accessibility notes
```

Then implement.
