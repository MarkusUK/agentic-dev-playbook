# MCP Design Exploration Prompt

```text
I like Direction [A/B/C], especially:

- [specific thing]
- [specific thing]
- [specific thing]

Now use available design MCP tools to explore this direction.

Use:
- Figma MCP if useful
- Stitch MCP if useful
- 21st.dev/Magic only for component inspiration, not final design authority
- Rive only for motion/animation ideas if relevant

Do not write production code yet.

Generate 3 screen concepts for:

1. Onboarding
2. Home/dashboard
3. Main detail screen
4. Create/edit flow
5. Settings/profile

For each screen:
- describe layout
- list components
- define colours/tokens used
- define states
- define empty/loading/error states
- define accessibility notes
- identify reusable components

Then produce a brand kit containing:

- logo concept
- colour tokens
- typography scale
- spacing scale
- radius scale
- shadow/elevation rules
- icon style
- illustration style
- motion principles
- light mode
- dark mode
- platform implementation notes
- design risks to avoid

Keep the result implementation-ready, but do not write production code yet.
```
