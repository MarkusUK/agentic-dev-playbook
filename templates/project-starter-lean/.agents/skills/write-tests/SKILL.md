# Write Tests Skill

When invoked:

1. Inspect changed code and nearby tests.
2. Identify the behaviour being tested.
3. Match existing test style, fixtures, mocks, and naming.
4. Add regression tests for bug fixes.
5. Cover success, failure, and edge cases where reasonable.
6. Run the smallest relevant test command first.
7. Do not delete or weaken tests to make the suite pass.

End with:

```text
Tests added:
Commands run:
Remaining gaps:
```
