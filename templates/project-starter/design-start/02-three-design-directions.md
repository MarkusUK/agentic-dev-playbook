# 02 - Three Design Directions

Use this after the brand discovery brief.

## Copy/paste prompt

```text
Based on the product idea and brand discovery brief, create 3 distinct design directions.

For each direction include:

- Name
- One-sentence concept
- Brand personality
- Colour palette with hex values
- Typography direction
- Spacing/layout style
- Component style
- Icon/illustration style
- Motion/animation style
- Example home screen description
- Example onboarding screen description
- Example settings/profile screen description
- Pros
- Cons
- Best fit
- Risks/clichés to avoid

Then compare the 3 directions in a table.

Finally, recommend one direction and explain why.

Do not write code yet.
Do not use external MCP tools yet unless I explicitly ask you to.
```

## Example directions for a Japanese sakura theme

```text
1. Minimal Sakura
   Soft, calm, lots of whitespace, pale blossom accents.

2. Kyoto Editorial
   Premium, refined, inspired by Japanese print/layout traditions, but modern.

3. Night Hanami
   Dark mode-first, lantern glow, deep indigo, sakura highlights, elegant motion.
```

## Avoid

```text
- too much pink
- anime clichés unless that is the intended audience
- decorative Japanese motifs used randomly
- low-contrast pastel text
- mixing too many typefaces
- busy blossom backgrounds
- cultural references without purpose
```
